About Me

I'm a passionate software engineer with a background that combines technical expertise with creative problem-solving. Currently working freelance and at my own company, Sanctuary DSP, I develop diverse software solutions ranging from web and mobile applications to specialised cross-platform, low-latency audio software using modern C++, Go, Python, and JavaScript.

My journey spans from music performance and production to full-stack software development, giving me a deep understanding of both the technical and artistic sides of technology. I'm particularly passionate about real-time DSP, neural audio processing, and creating tools that empower musicians and audio professionals, while also delivering robust web applications, desktop software, mobile apps, and enterprise solutions across various industries.

When I'm not coding client projects, you'll find me contributing to open-source initiatives across multiple domains — from audio software projects such as collaborating with the Surge Synthesizer team, to developing CI/CD migration tools and general-purpose software that solve real-world problems for developers and creators alike.
